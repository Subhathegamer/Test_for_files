1. Basic Document Structure

These essential tags form the skeleton of every HTML file:

  • <!DOCTYPE html>: Declares the document as HTML5.
  • <html>: The root element that wraps all content.
  • <head>: Contains metadata like the page title and links to CSS.
  • <title>: Defines the text shown in the browser tab.
  • <body>: Contains all visible content like text and images.

2. Text & Content Formatting

Used to organize and style text directly:

  • <h1> to <h6>: Headings, with h1 being the largest/most important.
  • <p>: Defines a paragraph.
  • <br>: Inserts a single line break.
  • <hr>: Creates a horizontal rule to separate content.
  • <strong> / <b>: Makes text bold (strong is for semantic importance).
  • <em> / <i>: Makes text italic (em is for emphasis).
  • <blockquote>: Used for long quotations from another source.
  • <pre>: Displays preformatted text, preserving spaces and line breaks.

3. Links & Multimedia

Used for navigation and embedding external media:

  • <a>: Creates a hyperlink using the href attribute.
  • <img>: Embeds an image (uses src and alt attributes).
  • <audio>: Embeds sound content.
  • <video>: Embeds video content.
  • <source>: Specifies multiple media resources for audio or video.

4. Lists

Used to group related items together:

  • <ul>: Creates an unordered (bulleted) list.
  • <ol>: Creates an ordered (numbered) list.
  • <li>: Defines a single item within a list.
  • <dl>, <dt>, <dd>: Used for description lists (term and definition).

5. Tables

Used for presenting data in rows and columns:

  • <table>: The container for the table.
  • <tr>: Defines a table row.
  • <th>: Defines a header cell (usually bold and centered).
  • <td>: Defines a standard data cell.

6. Forms & User Input

Used to collect data from users:

  • <form>: The container for an interactive form.
  • <input>: Creates various input fields (text, checkbox, radio, etc.).
  • <button>: Defines a clickable button.
  • <textarea>: For multi-line text input.
  • <label>: Provides a caption for an input element.
  • <select>, <option>: Creates a dropdown selection list.

7. Layout & Semantic Tags

These define the meaning and structure of different page sections:

  • <div>: A generic container for grouping elements (block-level).
  • <span>: A generic container for inline text.
  • <header>, <footer>: Define the top and bottom sections of a page.
  • <nav>: Used specifically for navigation links.
  • <section>, <article>, <aside>: Define independent content areas.
